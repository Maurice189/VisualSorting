Abstract

Comb sort is an improvement of bubble sort. In bubble sort the only adjacent pair of elements are compared (and possibly swapped) whereas in comb sort elements at a variable distance (gap size) can be compared and swapped. Though this gap size is not static - it starts with the largest possible gap size and shrinks continuously by certain factor. This behaviour eliminates turtles (small values near the end of the list) since in a bubble sort these slow the sorting down tremendously. (Rabbits, large values around the beginning of the list, do not pose a problem in bubble sort)
