Abstract

Merge sort is a divide and conquer algorithm that was invented by John von Neumann (an incredibly smart human being) in 1945 - it is an O(n log n) comparison-based sorting algorithm. Most implementations produce a stable sort. Merge sort performs equally sized splits of a list util there are only lists of 1-element left - this is the divide step. Next those sub lists are repeatedly merged to greater (sorted) sub lists until the there is one list over - this is the conquer step. Merge sort is not an in-place algorithm (all lists has to be newly allocated), which means it is not cache friendly. Therefore it is typically slower than quick sort. However it is especially suited for slow-access media (like external disk or tapes) and for sorting linked lists.
